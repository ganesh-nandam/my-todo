# 📝 Interactive To-Do List WebApp

A sleek, responsive, and dynamic To-Do List application built with pure HTML, CSS, and JavaScript. This project features full CRUD functionality, real-time data persistence, and a modern UI with dynamic priority tagging and statistic tracking.

Live demo: 

## 🚀 Features

* **Full CRUD Functionality:** Create, Read, Update, and Delete tasks seamlessly.
* **Dynamic Greetings:** Automatically parses the time of day to greet the user dynamically.
* **Priority & Status Management:** Color-coded priority tags and status badges to easily organize tasks.
* **Live Statistics:** Displays real-time completion status (e.g., *"You've got 4 tasks today"*).
* **Local Storage Persistence:** Your tasks are saved automatically in the browser's `localStorage`, so you never lose your data on refresh.
* **Responsive Layout:** Built using modern CSS Grid and Flexbox structures to ensure it looks great on mobile, tablet, and desktop screens.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (CSS Grid & Flexbox)
* **Scripting:** Vanilla JavaScript (ES6+)
* **Icons:** Font Awesome CDN
